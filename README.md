# Advanced Web Scraper

Крутой и гибкий инструмент для извлечения данных с веб-сайтов.

## Особенности

- Сканирование одного или множества URL.
- Извлечение данных по CSS-селекторам.
- Поддержка сохранения в CSV, JSON, Excel.
- Ротация User-Agent.
- Задержка между запросами для этичности.
- Цветной вывод в терминале.
